为什么叫 PepperBot？#

  • 我喜欢吃农家小炒肉(辣椒炒肉)

后端/协议端#

  • 协议端，或者说无头 QQ，无头微信之类的东西，实现了与 QQ、微信等平台的直接通信(接受并发送 QQ 消息的能力)
  • PepperBot 本身并没有实现与 QQ、微信、TG 等平台通信的能力，只负责处理“业务逻辑”
  • PepperBot 通过与协议端的通信，实现了与平台的通信
  • QQ 基于 OneBot 协议，所以所有符合 OneBot 标准的“协议端”，都可以使用，一般建议使用 go-cqhttp 作为“后端”
  • 微信基于可爱猫
