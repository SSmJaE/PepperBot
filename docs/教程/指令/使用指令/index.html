使用指令

在BotRoute中注册​

和事件响应中的handler一样，我们也需要将指令，通过BotRoute注册之后，才能跑起来

配置和handler类似

@as_command()
class MyCommand:
    pass

bot.apply_routes([
    BotRoute(
        friends=None,
        groups={
            "onebot": ["123456"]
        },
        handlers=[],
        commands = [MyCommand],
    )
])
